# 基于 STM32 的多功能智能风扇控制系统

> 嵌入式课程设计 / 电子竞赛作品 —— 以 STM32F103 为主控，实现直流电机的多模式调速、RTC 定时启停、秒表计时与人机交互功能。

---

## 📋 项目简介

本项目是一款面向智能家居场景的**多功能风扇嵌入式控制系统**。系统以 STM32F103 为核心控制器，集成 RTC 实时时钟、PWM 电机调速、ADC 采集、LCD 液晶显示与按键人机交互等模块，支持手动多档调速、旋钮无级调速、双模式自然风模拟以及定时自动启停功能，具备良好的实用性与扩展性。

---

## ✨ 功能特性

| 功能模块 | 说明 |
|---------|------|
| **🕐 RTC 实时时钟** | 支持 12/24 小时制切换，可手动校准时间 |
| **⏱️ 秒表计时** | 基于 TIM3 中断实现精准秒表，支持启动、暂停、复位 |
| **⏰ 定时启停** | 可独立设置风扇自动开启时间与关闭时间，到达预设时间后蜂鸣器 + LED 声光提示并自动执行 |
| **🎚️ 三档定速** | 按键切换高速（60%）、中速（40%）、低速（10%） |
| **🔄 无级调速** | 通过 ADC 采集旋钮电位器电压，实现 PWM 占空比 0~100% 连续可调 |
| **🍃 自然风模式 1** | 模拟自然风感：风速由弱到强再到弱周期性循环变化 |
| **🍃 自然风模式 2** | 分段式自然风：先高速运行 5s，再渐变减速，再低速运行 5s，循环往复 |
| **🖥️ LCD 人机交互** | 2.8/3.5 寸 TFTLCD 实时显示时间、模式、风速百分比及操作菜单 |
| **🔊 声光提示** | 定时触发或按键操作时，蜂鸣器与 LED 提供即时反馈 |

---

## 🛠️ 硬件平台

- **主控芯片**：STM32F103ZET6（STM32F1 系列）
- **开发板**：正点原子战舰/精英 STM32F103 开发板（兼容）
- **电机驱动**：直流减速电机 + L298N / MOS 驱动模块（PWM 调速）
- **显示模块**：2.8/3.5 寸 TFTLCD（ILI9341/NT35310 驱动，分辨率 320×240）
- **输入设备**：矩阵按键 / 独立按键（KEY0~KEY4）+ ADC 旋钮（电位器接 PA8/ADC_Channel_8）
- **RTC 时钟**：STM32 内置 RTC + 外部 32.768kHz 晶振
- **其他外设**：蜂鸣器（PB8）、LED0（PB5）

## 🎮 使用说明 / 按键定义

上电后，LCD 显示主菜单界面，通过按键进入对应功能：

| 按键 | 功能 |
|------|------|
| **KEY 1** | 进入**时间校准**模式（时+/时-/分+/分-/秒+/秒-） |
| **KEY 2** | **切换 12/24 小时制** |
| **KEY 3** | 进入**秒表**模式（启动 / 暂停 / 复位 / 退出） |
| **KEY 4** | 设置**定时启动时间** |
| **KEY 5** | 设置**定时关闭时间** |
| **KEY 6** | **三档定速**调速（高速 / 中速 / 低速） |
| **KEY 7** | **旋钮无级调速**模式（ADC 采集，实时调节） |
| **KEY 8** | 进入**自然风模式 1**（慢-快-慢循环） |
| **KEY 9** | 进入**自然风模式 2**（分段高低速交替） |
| **KEY 10** | **紧急停止**电机（PWM 输出关闭） |
| **KEY 16** | 退出当前子模式，返回主界面 |

> 各子模式下，LCD 会实时显示当前风速百分比、运行状态及 RTC 时间。

## 📌 核心模块技术要点

### 1. PWM 调速（TIM1）
- 使用 TIM1 通道 3（CH3）输出 PWM，频率 10kHz
- 自动重装载值 ARR = 719，预分频 PSC = 0，对应 72MHz 主频
- 占空比通过 `TIM_SetCompare3()` 动态修改：
  - 三档固定占空比：60% / 40% / 10%
  - 无级调速：`pwm_val = adc_val * 1000 / 4095`，映射 ADC 12bit 值到 PWM 脉宽

### 2. RTC 定时启停
- 利用 STM32 内置 RTC + 外部低速晶振，提供秒级精准走时
- 可分别设定 `start_time` 与 `stop_time`
- 主循环每秒比对当前 RTC 时间与设定时间，触发蜂鸣器 + LED + 电机启停

### 3. 自然风算法
- **模式 1**：for 循环递减/递增 PWM 占空比，配合 200ms 延时，形成平滑呼吸灯式风速曲线
- **模式 2**：利用 TIM2 定时器中断计数 5s，实现“高速维持→渐变减速→低速维持→渐变加速”的周期性自然风

### 4. ADC 旋钮采集
- ADC1 通道 8（PA8）采集电位器电压
- 采用 10 次采样平均滤波，消除抖动
- 12bit 分辨率（0~4095）线性映射至 PWM 占空比 0~100%

## 🚀 快速开始

1. **克隆仓库**
   ```bash
   git clone https://github.com/Ziran-OwO/multi-function-fan.git
   ```

2. **导入工程**
   - 将 `main.c` 替换到你本地正点原子 STM32F1 工程模板的 `USER` 目录下
   - 确保已添加 `HARDWARE` 与 `SYSTEM` 目录下的所有驱动文件到 Keil 工程

3. **硬件连接**
   - 电机 PWM 输入接 **PA10**（TIM1_CH3）
   - 电位器输出接 **PA8**（ADC1_IN8）
   - 按键接 **PE0~PE4**（根据具体开发板调整）
   - LCD 接 **FSMC** 接口（正点原子默认接线）

4. **编译下载**
   - Keil 中选择 `STM32F103ZE` 芯片型号
   - 编译无误后，通过 ST-Link 下载到开发板
